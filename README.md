# tweepy_follower

 ## Install 
 pandasのインストール
```
pip install pandas
```
 tweepyのインストール
```
pip install tweepy
```
## Usage
 ` getFollowersAccount.py ` を開き、
 https://github.com/tnkshuuhei/tweepy_follower/blob/2b79167eb3a4d996a8090c968209e80291b97d16/getFollowersAccount.py#L15
 `"@以降のユーザーIDを入力"`  に、取得したいTwitterアカウントIDを入力。
 保存し、実行。
